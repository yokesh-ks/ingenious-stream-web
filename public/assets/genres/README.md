# Genre Images

This directory contains images for radio genre cards.

## Required Images

Place the following images in this directory (recommended size: 800x800px, format: JPG):

### Main Genres
- `pop.jpg` - Pop music imagery
- `rock.jpg` - Rock music imagery
- `jazz.jpg` - Jazz music imagery
- `classical.jpg` - Classical music/orchestra imagery

### Default
- `default.jpg` - Default fallback image for genres without specific images

## Image Guidelines

- **Dimensions**: 800x800px (1:1 aspect ratio)
- **Format**: JPG or WebP
- **Size**: Keep under 200KB per image for optimal loading
- **Quality**: High quality but optimized for web
- **Style**: Consistent visual style across all images
- **Content**: Representative imagery for each genre
